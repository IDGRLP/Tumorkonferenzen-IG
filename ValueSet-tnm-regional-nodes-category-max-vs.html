ValueSet: TNM Regional Nodes Maximum Value Set

Official URL: http://idg-rlp.de/fhir/tumorkonferenzen/ValueSet/tnm-regional-nodes-category-max-vs Version: 0.1.0
Draft as of 2024-05-28 Computable Name: TNMRegionalNodesCategoryMaxVS

Values for TNM regional nodes category must be selected from this value set, which includes all codes from UICC and all codes from TNMRegionalNodesCategoryVS.

References

This value set is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Import all the codes that are contained in TNM Regional Nodes Category Value Set
  • Include all codes defined in https://www.uicc.org/resources/tnm
